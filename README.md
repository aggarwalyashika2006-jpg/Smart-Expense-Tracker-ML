## 📌 Overview

The Smart Expense Tracker is a data-driven application designed to help users track, analyze, and predict their spending behavior.

It combines **data analysis, visualization, and machine learning** to provide meaningful financial insights through an interactive dashboard.

---

## ❗ Problem Statement

Many individuals struggle to manage their daily expenses due to lack of proper tracking and insights.
As a result, overspending and poor financial planning become common issues.

---

## 💡 Solution

This project solves the problem by:

* Tracking expenses in structured format
* Visualizing spending patterns
* Providing category-wise and monthly insights
* Predicting future expenses using Machine Learning

---

## 🚀 Features

### 📊 Interactive Dashboard

* Clean and user-friendly interface using Streamlit
* Real-time data filtering

### 📂 Expense Analysis

* Category-wise spending analysis
* Monthly trend visualization

### 🔍 Smart Filtering

* Filter expenses based on category
* Dynamic data updates

### 🔮 ML-Based Prediction

* Predicts future spending using Linear Regression
* Helps users plan finances better

### 💰 Budget Alert System

* Alerts users when spending exceeds budget
* Helps maintain financial discipline

### 📌 Insights Generation

* Identifies top spending category
* Highlights average and maximum spending

---

## 🛠 Tech Stack

* **Python**
* **Pandas, NumPy** (Data Processing)
* **Plotly** (Visualization)
* **Streamlit** (Dashboard)
* **Scikit-learn** (Machine Learning)

---

## ⚙️ How It Works

1. Expense data is loaded from dataset
2. Data is cleaned and processed using Pandas
3. Spending patterns are analyzed
4. Visualizations are created using Plotly
5. ML model predicts future expenses
6. Dashboard displays insights interactively
