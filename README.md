# 💸 Smart Expense Tracker (Data Science Project)

## 📌 Overview
This project is an advanced Expense Tracker built using Python, Data Science, and Machine Learning.

It allows users to:
- Track expenses
- Analyze spending patterns
- Visualize financial data
- Predict future expenses

---

## 🚀 Features
- 📊 Interactive Dashboard (Streamlit)
- 📂 Category-wise Analysis
- 📅 Monthly Trends
- 💳 Payment Method Insights
- 🔮 ML-based Expense Prediction
- ⚠️ Overspending Detection

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Plotly
- Streamlit
- Scikit-learn
